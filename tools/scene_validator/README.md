# SceneValidator

A tool for validating scene structure and composition in media production.

## Overview

SceneValidator helps ensure technical and creative consistency in media production by validating scene elements against defined rules and standards.

See the main documentation in the `docs` directory for detailed usage instructions.

## Quick Start

```bash
# Install requirements
pip install -r requirements.txt

# Run validation on a sample scene
python scene_validator.py --scene-file ../../examples/scene_validator/sample_scene.json
```
